# Simple Poker - Unity

 This project is a simple singleplayer poker with complete game mechanics and opponent ai.
 
## Unity Version: 

* 2021.3.0f1

## JetBrains Rider: 

* 2022.1.2

## Assets Used: 

* UniTask

* UniTask PubSub

## Instruction

* Play Game Scene
* Wait for your turn
* Click on displayed buttons to perform action
